# Log Anomaly Detection using Deep Learning

## Project Overview
This project implements four deep learning approaches (CNN, LSTM, BiLSTM, Autoencoder) for log anomaly detection on HDFS dataset. The CNN model achieved 94% accuracy.

## Models
- CNN: Best performance (94% accuracy)
- LSTM: Good for sequential patterns
- BiLSTM: Bidirectional context
- Autoencoder: Unsupervised baseline

## Results
- CNN accuracy: 94%
- Improved anomaly recall by 15% using class weights
